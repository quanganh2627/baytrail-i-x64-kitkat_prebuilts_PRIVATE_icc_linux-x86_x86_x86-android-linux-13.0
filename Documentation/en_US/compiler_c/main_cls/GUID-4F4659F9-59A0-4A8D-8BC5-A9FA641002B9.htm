_writefsbase_u32(), _writefsbase_u64()

Write the value to the FS base register.

Syntax

extern void _writefsbase_u32(int 32 val);

extern void _writefsbase_u64(unsigned __int64 val);

Arguments

val

the value to be written to the FS base register

Description

These intrinsics write the given value to the FS segment base register. Both intrinsics are mapped to WRFSBASE instruction.
