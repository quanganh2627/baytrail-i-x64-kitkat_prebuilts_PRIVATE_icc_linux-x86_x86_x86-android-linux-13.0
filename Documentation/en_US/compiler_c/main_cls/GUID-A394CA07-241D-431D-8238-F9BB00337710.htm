Intel® C++ Compiler XE 13.1 User and Reference Guides

vd

Disable or enable hidden vtordisp field in C++ objects.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

None

Windows:

/vdval

Arguments

val

Possible values are:

0

Disables hidden vtordisp field in C++ objects.

1

Enables hidden vtordisp field in C++ objects.

Default

/vd1

The compiler enables hidden vtordisp field in C++ objects.

Description

This option disables or enables hidden vtordisp field in C++ objects.

Alternate Options

None
