    Multiply and add int32 vectors. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_fmadd233_epi32/ _mm512_mask_fmadd233_epi32
    Multiply and add int32 vectors with alternating elements. The corresponding instruction is VPMADD233D. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

  • _mm512_fmsub_pd/ _mm512_fmsub_round_pd/ _mm512_mask_fmsub_pd/ _mm512_mask_fmsub_round_pd/ _mm512_mask3_fmsub_pd/ _mm512_mask3_fmsub_round_pd

  • _mm512_fnmsub_ps/ _mm512_fnmsub_round_ps/ _mm512_mask_fnmsub_ps/ _mm512_mask3_fnmsub_ps/ _mm512_mask_fnmsub_round_ps/ _mm512_mask3_fnmsub_round_ps
    Multiply, negate, and subtract float32 vectors. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_fnmsub_pd/ _mm512_fnmsub_round_pd/ _mm512_mask_fnmsub_pd/ _mm512_mask3_fnmsub_pd/ _mm512_mask_fnmsub_round_pd/ _mm512_mask3_fnmsub_round_pd
    Multiply, negate, and subtract float64 vectors. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mulhi_epi32/ _mm512_mask_mulhi_epi32
    Multiply int32 vectors and store the high half of the result. The corresponding instruction is VPMULHD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mulhi_epu32/ _mm512_mask_mulhi_epu32
    Multiply uint32 vectors and store the high half of the result. The corresponding instruction is VMULHPU. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mullo_epi32/ _mm512_mask_mullo_epi32
    Multiply int32 vectors and store the low half of the result. The corresponding instruction is VMULLPI. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mul_pd/ _mm512_mask_mul_pd
    Multiply float64 vectors. The corresponding instruction is VMULPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mul_ps/ _mm512_mask_mul_ps
    Multiply float32 vectors. The corresponding instruction is VMULPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mul_round_pd/ _mm512_mask_mul_round_pd
    Multiplies rounded float64 vectors. The corresponding instruction is VMULPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_mul_round_ps/ _mm512_mask_mul_round_ps
    Multiplies rounded float32 vectors. The corresponding instruction is VMULPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_scale_ps/ _mm512_mask_scale_ps
    Scales float32 vectors. Corresponding instruction is VSCALEPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_scale_round_ps/ _mm512_mask_scale_round_ps
    Scales rounded float32 vectors. Corresponding instruction is VSCALEPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
