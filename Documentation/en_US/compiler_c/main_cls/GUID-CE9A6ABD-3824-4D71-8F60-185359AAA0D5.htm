_mm512_cvtepu32lo_pd/ _mm512_mask_cvtepu32lo_pd

Converts uint32 vector to float64 vector. Corresponding instruction is VCVTUDQ2PD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

Without Mask

extern __m512d __cdecl _mm512_cvtepu32lo_pd(__m512i v2);

With Mask

extern __m512i __cdecl _mm512_mask_cvtepu32lo_pd(_m512d v1_old, __mmask8 k1, __m512i v2);

Parameters

v2

 float64 vector used for the conversion operation

v1_old

 Source vector that retains old values of the destination vector; the resulting vector gets corresponding elements from v1_old for zero mask bits

k1

 Writemask; only those elements of the source vectors with corresponding bit set to '1' in the k1 mask are computed and stored in the result; elements in the result vector corresponding to zero bit in k1 are copied from corresponding elements of vector v1_old

Description

Performs an element-by-element conversion of the unsigned int32 vector v2 to a float64 vector. The resulting elements are written into the lower half of the result vector. The remaining locations (upper half of the result vector) are set to '0'.

The masked variant has one additional argument: k1. Only those elements in the source vectors with the corresponding bit set in vector mask k1 are used for computing.

Returns

Returns the result of the conversion operation.
