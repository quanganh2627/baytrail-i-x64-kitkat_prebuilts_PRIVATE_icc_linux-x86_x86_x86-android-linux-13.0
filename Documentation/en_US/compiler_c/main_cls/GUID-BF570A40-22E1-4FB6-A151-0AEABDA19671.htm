Intel® C++ Compiler XE 13.1 User and Reference Guides

Getting Help and Support

The Intel® C++ Compiler lets you build and optimize C/C++ applications for the Linux* OS (operating system). You can use the compiler on the command line or in the Eclipse* integrated development environment.

This document assumes that you are familiar with the C++ programming language and with the architecture requirements for the application you are developing. You should also be familiar with the operating system on your host computer.
