T

Tells the linker to read link commands from a file.

IDE Equivalent

None

Architectures

All

Syntax

Linux:

-Tfilename

OS X:

None

Windows:

None

Arguments

filename

Is the name of the file.

Default

OFF

The linker does not read link commands from a file.

Description

This option tells the linker to read link commands from a file.

Alternate Options

None
