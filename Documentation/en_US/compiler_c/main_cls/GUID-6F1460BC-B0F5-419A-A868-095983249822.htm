Gd

Makes __cdecl the default calling convention.

IDE Equivalent

Windows: Advanced > Calling Convention

Linux: None

OS X: None

Architectures

IA-32 architecture

Syntax

Linux and OS X:

None

Windows:

/Gd

Arguments

None

Default

ON

The default calling convention is __cdecl.

Description

This option makes __cdecl the default calling convention.

Alternate Options

None
