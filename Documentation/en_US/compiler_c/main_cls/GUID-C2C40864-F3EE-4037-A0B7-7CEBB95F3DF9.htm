Intel® C++ Compiler XE 13.1 User and Reference Guides

dumpversion

Displays the version number of the compiler.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-dumpversion

Windows:

None

Arguments

None

Default

OFF

The compiler does not display the compiler version number.

Description

This option displays the version number of the compiler. It does not compile your source files.

Alternate Options

None

Example

Consider the following command:

icc -dumpversion

If the above is specified when using the Intel® C++ Compiler XE 13.0, the compiler displays "13.0".
