Intel® C++ Compiler XE 13.1 User and Reference Guides

Error Function Operations

Parent topic: Short Vector Math Library Intrinsics
  • _mm512_cdfnorminv_pd, _mm512_mask_cdfnorminv_pd
    Calculates the inverse cumulative distribution value. Vector variant of cdfnorminv(x) function for a float64 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cdfnorminv_ps, _mm512_mask_cdfnorminv_ps
    Calculates the inverse cumulative distribution value. Vector variant of cdfnorminv(x) function for a float32 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erf_pd/ _mm512_mask_erf_pd
    Calculates error function. Vector variant of erf(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erf_ps/ _mm512_mask_erf_ps
    Calculates error function. Vector variant of erf(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erfc_pd/ _mm512_mask_erfc_pd
    Calculates complementary error function. Vector variant of erfc(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erfc_ps/ _mm512_mask_erfc_ps
    Calculates complementary error function. Vector variant of erfc(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erfinv_pd/ _mm512_mask_erfinv_pd
    Calculates inverse error function. Vector variant of erfinv(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_erfinv_ps/ _mm512_mask_erfinv_ps
    Calculates inverse error function. Vector variant of erfinv(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
