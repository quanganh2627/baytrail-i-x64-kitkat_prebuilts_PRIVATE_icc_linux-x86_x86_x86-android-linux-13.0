Intel® C++ Compiler XE 13.1 User and Reference Guides

Addition Operations

Parent topic: Arithmetic Intrinsics
  • _mm512_add_epi32/ _mm512_mask_add_epi32
    Adds int32 vectors. The corresponding instruction is VPADD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_add_epi64/ _mm512_mask_add_epi64
    Adds int64 vectors. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_add_ps/ _mm512_mask_add_ps
    Adds float32 vectors. The corresponding instruction is VADDPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_add_pd/ _mm512_mask_add_pd
    Adds float64 vectors. The corresponding instruction is VADDPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addn_ps/ _mm512_mask_addn_ps
    Adds and negates sum of float32 vectors. The corresponding instruction is VADDNPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addn_pd/ _mm512_mask_addn_pd
    Adds and negates the sum of float64 vectors.  The corresponding instruction is VADDNPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addsets_epi32/ _mm512_mask_addsets_epi32
    Adds int32 vectors and sets mask to sign. The corresponding instruction is VPADDSETSD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addsets_ps/ _mm512_mask_addsets_ps
    Adds float32 vectors and sets mask to sign. The corresponding instruction is VADDSETSPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addsetc_epi32/ _mm512_mask_addsetc_epi32
    Adds int32 vectors and sets mask to carry. The corresponding instruction is VPADDSETCD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_adc_epi32/ _mm512_mask_adc_epi32
    Adds int32 vectors with carry. The corresponding instruction is VPADCD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addn_round_pd/ _mm512_mask_addn_round_pd
    Adds and negates rounded sum of float64 vectors. The corresponding instruction is VADDNPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addn_round_ps/ _mm512_mask_addn_round_ps
    Adds and negates rounded sum of float32 vectors. The corresponding instruction is VADDNPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_add_round_pd/ _mm512_mask_add_round_pd
    Adds rounded float64 vectors. The corresponding instruction is ADDPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_add_round_ps/ _mm512_mask_add_round_ps
    Adds rounded float32 vectors. The corresponding instruction is VADDPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_addsets_round_ps/ _mm512_mask_addsets_round_ps
    Adds rounded float32 vectors and sets mask to sign. The corresponding instruction is VADDSETSPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
