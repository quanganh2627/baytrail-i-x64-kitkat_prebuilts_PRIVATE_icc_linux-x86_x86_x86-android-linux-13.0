fno-rtti

Disables support for run-time type information (RTTI).

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-fno-rtti

Windows:

None

Arguments

None

Default

OFF

Description

This option disables support for run-time type information (RTTI).

Alternate Options

None
