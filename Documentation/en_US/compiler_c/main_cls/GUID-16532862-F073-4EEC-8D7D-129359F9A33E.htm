C99 Macros

Many routines in the Intel math library are more highly optimized for Intel® microprocessors than for non-Intel microprocessors.

The Intel math library and mathimf.h header file support the following C99 macros:

int fpclassify(x);

int isfinite(x);

int isgreater(x, y);

int isgreaterequal(x, y);

int isinf(x);

int isless(x, y);

int islessequal(x, y);

int islessgreater(x, y);

int isnan(x);

int isnormal(x);

int isunordered(x, y);

int signbit(x);
