Overview: Intel® Streaming SIMD Extensions 3 (Intel® SSE3)

The Intel® C++ intrinsics listed in this section are designed for the Intel® Pentium® 4 processor with Streaming SIMD Extensions 3 (Intel® SSE3). The new Intel® SSE3 intrinsics include:

  • Floating-point Vector Intrinsics

  • Integer Vector Intrinsics

  • Miscellaneous Intrinsics

  • Macro Functions

The prototypes for these intrinsics are in the pmmintrin.h header file.

Note

You can also use the single ia32intrin.h header file for any intrinsics based on IA-32 architecture.
