_mm256_cvtph_ps()

Converts eight half-precision (16-bit) floating point values to single-precision floating point values.

Syntax

extern __m256 _mm_cvtph_ps(__m128i x, int imm);

Arguments

X

a vector containing eight 16-bit FP values

Imm

A conversion control constant

Description

This intrinsic converts eight half-precision (16-bit) floating point values to single-precision floating point values. The corresponding Intel® Post 32nm processor extension instruction is VCVTPH2PS.

Returns

A vector containing eight single-precision floating point elements.
