Intel® C++ Compiler XE 13.1 User and Reference Guides

tbb, Qtbb

Tells the compiler to link to the Intel® Threading Building Blocks (Intel® TBB) libraries.

IDE Equivalent

Windows: None

Linux: Performance Library Build Components > Use Intel® Threading Building Blocks Library

OS X: Libraries > Use Intel® Threading Building Blocks Library

Architectures

All

Syntax

Linux and OS X:

-tbb

Windows:

/Qtbb

Arguments

None

Default

OFF

The compiler does not link to the Intel® TBB libraries.

Description

This option tells the compiler to link to the Intel® Threading Building Blocks (Intel® TBB) libraries and include the Intel® TBB headers.

Alternate Options

None
