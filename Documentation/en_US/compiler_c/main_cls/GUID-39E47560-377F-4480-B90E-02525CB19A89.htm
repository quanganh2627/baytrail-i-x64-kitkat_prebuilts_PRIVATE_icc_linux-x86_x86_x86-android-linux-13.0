Porting Linux* OS or OS X* Code to Windows* OS

You can use the /Qgcc-dialect option to port Linux* OS or OS X* code to Windows* OS. This option enables support for a language dialect that is compatible with GNU* gcc, while maintaining link compatibility with Microsoft Windows*.
