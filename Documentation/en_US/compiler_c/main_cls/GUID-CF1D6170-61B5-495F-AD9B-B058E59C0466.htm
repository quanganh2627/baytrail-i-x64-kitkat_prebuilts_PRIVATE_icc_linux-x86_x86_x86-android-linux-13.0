Intel® C++ Compiler XE 13.1 User and Reference Guides

Conversion Intrinsics

Parent topic: Vector Intrinsics
  • _mm512_cvtpd_pslo/ _mm512_mask_cvtpd_pslo
    Converts float64 vector to float32 vector. Corresponding instruction is VCVTPD2PS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvt_roundpd_pslo/ _mm512_mask_cvt_roundpd_pslo
    Converts float64 vector to float32 vector. Corresponding instruction is VCVTPD2PS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvt_roundpd_epi32lo/ _mm512_mask_cvt_roundpd_epi32lo
    Converts and rounds float64 vector to int32 vector. Corresponding instruction is VCVTFXPNTPD2DQ. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtfxpnt_roundpd_epu32lo/ _mm512_mask_cvtfxpnt_roundpd_epu32lo
    Converts and rounds float64 vector to uint32 vector. Corresponding instruction is VCVTFXPNTPD2UDQ. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtpslo_pd/ _mm512_mask_cvtpslo_pd
    Converts a float32 vector into a float64 vector. The corresponding instruction is VCVTPS2PD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtepu32lo_pd/ _mm512_mask_cvtepu32lo_pd
    Converts uint32 vector to float64 vector. Corresponding instruction is VCVTUDQ2PD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtepi32lo_pd/ _mm512_mask_cvtepi32lo_pd
    Converts int32 vector to float64 vector. Corresponding instruction is VCVTDQ2PD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtfxpnt_round_adjustepu32_ps/ _mm512_mask_cvtfxpnt_round_adjustepu32_ps
    Converts uint32 vector to float32 vector. Corresponding instruction is VCVTFXPNTUDQ2PS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtfxpnt_round_adjustps_epi32/ _mm512_mask_cvtfxpnt_round_adjustps_epi32
    Converts and rounds float32 vector to int32 vector. Corresponding instruction is VCVTFXPNTDQ2PS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtfxpnt_round_adjustps_epu32/ _mm512_mask_cvtfxpnt_round_adjustps_epu32
    Converts and rounds float32 vector to uint32 vector. Corresponding instruction is VCVTFXPNTPS2UDQ. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cvtfxpnt_round_adjustepu32_ps/ _mm512_mask_cvtfxpnt_round_adjustepu32_ps
    Converts uint32 vector to float32 vector. Corresponding instruction is VCVTFXPNTUDQ2PS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_round_ps/ _mm512_mask_round_ps
    Rounds float32 vectors. Corresponding instruction is VROUNDPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
