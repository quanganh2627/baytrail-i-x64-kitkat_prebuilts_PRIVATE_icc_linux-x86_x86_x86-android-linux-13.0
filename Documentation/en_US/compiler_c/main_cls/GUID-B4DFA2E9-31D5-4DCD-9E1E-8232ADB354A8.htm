_mm256_set1_epi32

Initializes 256-bit vector with scalar integer values. No corresponding Intel® AVX instruction.

Syntax

extern __m256i _mm256_set1_epi32(int);

Arguments

int

an integer32 value to be initialized into the 256-bit vector; there is one int parameter to initialize each integer32 vector element

Description

Initializes a 256-bit vector with scalar integer values (integer32 values) as specified by the int parameter.

Returns

An integer32 vector with all elements set to the specified scalar integer value.
