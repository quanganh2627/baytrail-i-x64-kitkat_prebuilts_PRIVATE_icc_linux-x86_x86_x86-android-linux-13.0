Intel® C++ Compiler XE 13.1 User and Reference Guides

Logarithmic Operations

Parent topic: Short Vector Math Library Intrinsics
  • _mm512_log10_pd/ _mm512_mask_log10_pd
    Calculates base-10 logarithm. Vector variant of log10(x) function for a float64 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log10_ps/ _mm512_mask_log10_ps
    Calculates base-10 logarithm. Vector variant of log10(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log2_pd/ _mm512_mask_log2_pd
    Calculates base-2 logarithm. Vector variant of log2(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log2_ps/ _mm512_mask_log2_ps
    Calculates base-2 logarithm. Vector variant of log2(x) function for a float32gument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log_pd/ _mm512_mask_log_pd
    Calculates natural logarithm. Vector variant of log(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log_ps/ _mm512_mask_log_ps
    Calculates natural logarithm. Vector variant of log(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_logb_pd, _mm512_mask_logb_pd
    Calculates signed exponent. Vector variant of logb(x) function for a float64 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_logb_ps, _mm512_mask_logb_ps
    Calculates signed exponent. Vector variant of logb(x) function for a float32 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

    Calculates natural logarithm. Vector variant of log1p(x) function for a float64 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_log1p_ps, _mm512_mask_log1p_ps
    Calculates natural logarithm. Vector variant of log1p(x) function for a float32 argument. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
