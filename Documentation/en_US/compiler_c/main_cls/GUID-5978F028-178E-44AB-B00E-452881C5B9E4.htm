_mm256_set_epi32

Initializes 256-bit vector with integer values. No corresponding Intel® AVX instruction.

Syntax

extern __m256i _mm256_set_epi32(int, int, int, int, int, int, int, int);

Arguments

int

an integer32 value to be initialized into the 256-bit vector; there are eight int parameters, one for each integer32 vector element

Description

Initializes a 256-bit vector with extended packed integer values (integer32 values) as specified by the int parameter.

Returns

An integer32 vector initialized with specified extended packed integer values.
