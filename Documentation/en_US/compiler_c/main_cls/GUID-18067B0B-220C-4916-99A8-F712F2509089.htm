Compare Operations between Floating Point Vectors

Parent topic: Compare Intrinsics
  • Compare Intrinsics for Floating Point Vectors
  • _mm512_cmp_pd_mask/ _mm512_mask_cmp_pd_mask
    Performs a comparison between float64 vectors. Corresponding instruction is VCMPPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cmp_ps_mask/ _mm512_mask_cmp_ps_mask
    Performs a comparison between float32 vectors. Corresponding instruction is VCMPPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
