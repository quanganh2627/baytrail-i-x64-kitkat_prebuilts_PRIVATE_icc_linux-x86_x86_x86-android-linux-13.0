_mm_cosd_ps, _mm256_cosd_ps

Calculates cosine value. Vector variant of cosd(x) function for a 128-bit/256-bit vector argument of float32 values.

Syntax

extern __m128 _mm_cosd_ps(__m128 v1);

extern __m256 _mm256_cosd_ps(__m256 v1);

Arguments

v1

vector with float32 values

Description

Calculates the cosine of vector v1 elements, measured in degrees.

Returns

128-bit/256-bit vector with the result of the operation.

Note

Many routines in the SVML library are more highly optimized for Intel® microprocessors than for non-Intel microprocessors.
