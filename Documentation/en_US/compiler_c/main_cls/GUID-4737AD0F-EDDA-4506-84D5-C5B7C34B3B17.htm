Wpragma-once

Determines whether a warning is issued about the use of #pragma once.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-Wpragma-once

-Wno-pragma-once

Windows:

None

Arguments

None

Default

-Wno-pragma-once

No warning is issued about the use of #pragma once.

Description

This option determines whether a warning is issued about the use of #pragma once.

Alternate Options

None
