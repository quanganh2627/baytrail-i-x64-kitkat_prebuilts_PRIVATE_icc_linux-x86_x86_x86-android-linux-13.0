Compare Operations between Integer Vectors

Parent topic: Compare Intrinsics
  • _mm512_cmp_epi32_mask/ _mm512_mask_cmp_epi32_mask
    Performs a comparison between int32 vectors. Corresponding instruction is VPCMPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cmp_epu32_mask/ _mm512_mask_cmp_epu32_mask
    Performs a comparison between uint32 vectors. Corresponding instruction is VPCMPUD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
