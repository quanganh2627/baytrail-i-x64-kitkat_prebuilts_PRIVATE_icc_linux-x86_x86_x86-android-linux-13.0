Intel® C++ Compiler XE 13.1 User and Reference Guides

Subtraction Operations

Parent topic: Arithmetic Intrinsics
  • _mm512_sub_epi32/ _mm512_mask_sub_epi32
    Subtracts int32 vectors. The corresponding instruction is   VPSUBD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sub_ps/ _mm512_mask_sub_ps
    Subtracts float32 vectors. The corresponding instruction is   VSUBPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sub_pd/ _mm512_mask_sub_pd
    Subtracts float64 vectors. The corresponding instruction is VSUBPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subr_epi32/ _mm512_mask_subr_epi32
    Reverse subtracts int32 vectors. The corresponding instruction is   VPSUBRD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subr_pd/ _mm512_mask_subr_pd
    Reverse subtracts float64 vectors. The corresponding instruction is   VSUBRPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subr_ps/ _mm512_mask_subr_ps
    Reverse subtracts float32 vectors. The corresponding instruction is   VSUBRPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sbbr_epi32/ _mm512_mask_sbbr_epi32
    Reverse subtracts int32 vectors with borrow. The corresponding instruction is VPSBBRD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subsetb_epi32/ _mm512_mask_subsetb_epi32
    Subtracts int32 vectors and sets borrow. The corresponding instruction is VPSUBSETBD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subrsetb_epi32/ _mm512_mask_subrsetb_epi32
    Reverse subtracts int32 vectors and sets borrow. The corresponding instruction is VPSUBRSETBD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sbb_epi32/_m512_mask_sbb_epi32
    Subtracts three int32 vectors with borrow. The corresponding instruction is VPSBBD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sub_round_pd/ _mm512_mask_sub_round_pd
    Subtracts rounded float64 vectors. The corresponding instruction is VSUBPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sub_round_ps/ _mm512_mask_sub_round_ps
    Subtracts rounded float32 vectors. The corresponding instruction is   VSUBPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subr_round_pd/ _mm512_mask_subr_round_pd
    Reverse subtracts rounded float64 vectors. The corresponding instruction is   VSUBRPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_subr_round_ps/ _mm512_mask_subr_round_ps
    Reverse subtracts rounded float32 vectors. The corresponding instruction is   VSUBRPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
