_mm512_castps_pd

Casts float32 vectors elements as float64 vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __m512d __cdecl _mm512_castps_pd(_m512 in);

Parameters

in

Vector used for casting operation

Description

Performs an element-by-element conversion/casting of float32 vector elements to float64 vector elements without any change in value.

Returns

Returns the result of the casting operation.
