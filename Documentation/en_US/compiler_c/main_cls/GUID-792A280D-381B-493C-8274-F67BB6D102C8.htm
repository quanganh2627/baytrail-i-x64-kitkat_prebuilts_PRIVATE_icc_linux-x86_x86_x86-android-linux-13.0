imacros

Allows a header to be specified that is included in front of the other headers in the translation unit.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-imacros filename

Windows:

None

Arguments

filename

Name of header file.

Default

OFF

Description

Allows a header to be specified that is included in front of the other headers in the translation unit.

Alternate Options

None
