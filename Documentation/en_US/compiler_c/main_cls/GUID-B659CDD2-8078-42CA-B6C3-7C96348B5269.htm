_mm512_kortestc

Performs a bitwise OR operation between two vector masks and sets CF flag. Corresponding instruction is KORTEST. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern int __cdecl _mm512_kortestc(__mmask16 k1, __mmask16 k2);  

Parameters

k1

 vector mask

k2

 vector mask

Description

Performs a bitwise OR operation between vector masks k2 and k1. If the result of the operation is all ones (1), the CF flag is set, otherwise not.

Returns

Returns a non-zero value if CF flag is set.
