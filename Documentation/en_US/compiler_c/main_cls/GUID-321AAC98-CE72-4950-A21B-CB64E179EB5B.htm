Square Root and Cube Root Operations

Parent topic: Short Vector Math Library Intrinsics
  • _mm512_sqrt_pd/ _mm512_mask_sqrt_pd
    Calculates square root value. Vector variant of sqrt(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_sqrt_ps/ _mm512_mask_sqrt_ps
    Calculates square root value. Vector variant of sqrt(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_invsqrt_pd/ _mm512_mask_invsqrt_pd
    Calculates inverse square root value. Vector variant of invsqrt(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_invsqrt_ps/ _mm512_mask_invsqrt_ps
    Calculates inverse square root value. Vector variant of invsqrt(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_hypot_pd/ _mm512_mask_hypot_pd
    Calculates hypotenuse value. Vector variant of hypot() function for a float64 arguments.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_hypot_ps/ _mm512_mask_hypot_ps
    Calculates hypotenuse value. Vector variant of hypot() function for a float32 arguments.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cbrt_pd/ _mm512_mask_cbrt_pd
    Calculates cube root value. Vector variant of cbrt(x) function for a float64 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_cbrt_ps/ _mm512_mask_cbrt_ps
    Calculates cube root value. Vector variant of cbrt(x) function for a float32 argument.   This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
