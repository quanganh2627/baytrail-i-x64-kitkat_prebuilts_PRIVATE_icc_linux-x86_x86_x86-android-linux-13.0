Intel® C++ Compiler XE 13.1 User and Reference Guides

_mm512_int2mask

Converts an integer value to a vector mask. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __mmask16__cdecl _mm512_int2mask(int mask);

Parameters

mask

 integer value

Description

Converts an integer value, mask, to a vector mask.

Returns

Returns an vector mask.
