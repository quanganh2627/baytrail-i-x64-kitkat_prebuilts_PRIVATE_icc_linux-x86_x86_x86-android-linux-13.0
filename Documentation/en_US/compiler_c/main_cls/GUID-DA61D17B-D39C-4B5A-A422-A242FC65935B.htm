Intel® C++ Compiler XE 13.1 User and Reference Guides

Reduction Intrinsics

Parent topic: Vector Intrinsics
  • Overview: Reduction Intrinsics
  • _mm512_reduce_add_ps/_mm512_mask_reduce_add_ps
    Returns a float32 scalar value of an addition operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_add_pd/_mm512_mask_reduce_add_pd
    Returns a float64 scalar value of an addition operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_add_epi32/_mm512_mask_reduce_add_epi32
    Returns an integer scalar value of an addition operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_and_epi32/_mm512_mask_reduce_and_epi32
    Returns an integer scalar value of a bitwise AND operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_or_epi32/_mm512_mask_reduce_or_epi32
    Returns an integer scalar value of a bitwise OR operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_[g]max_ps/_mm512_mask_reduce_[g]max_ps
    Determines the element with maximum scalar value in a float32 vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_[g]max_pd/_mm512_mask_reduce_[g]max_pd
    Determines the element with maximum scalar value in a float64 vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_max_epi32/_mm512_mask_reduce_max_epi32
    Determines the element with maximum scalar value in an integer vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_max_epu32/_mm512_mask_reduce_max_epu32
    Determines the element with maximum scalar value in an unsigned integer vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_[g]min_ps/_mm512_mask_reduce_[g]min_ps
    Determines the element with minimum scalar value in a float32 vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_[g]min_pd/_mm512_mask_reduce_[g]min_pd
    Determines the element with minimum scalar value in a float64 vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_min_epi32/_mm512_mask_reduce_min_epi32
    Determines the element with minimum scalar value in an integer vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_min_epu32/_mm512_mask_reduce_min_epu32
    Determines the element with minimum scalar value in an unsigned integer vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_mul_ps/_mm512_mask_reduce_mul_ps
    Returns a float32 scalar value of a multiplication operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_mul_pd/_mm512_mask_reduce_mul_pd
    Returns a float64 scalar value of a multiplication operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_reduce_mul_epi32/_mm512_mask_reduce_mul_epi32
    Returns an integer scalar value of a multiplication operation. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
