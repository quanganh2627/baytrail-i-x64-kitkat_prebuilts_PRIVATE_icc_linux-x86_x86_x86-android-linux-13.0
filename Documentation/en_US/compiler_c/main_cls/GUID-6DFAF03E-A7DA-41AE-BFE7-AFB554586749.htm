mregparm

Control the number registers used to pass integer arguments.

IDE Equivalent

None

Architectures

IA-32 architecture

Syntax

Linux and OS X:

-mregparm=value

Windows:

None

Arguments

None

Default

OFF

The compiler does not use registers to pass arguments.

Description

Control the number registers used to pass integer arguments.

Alternate Options

None
