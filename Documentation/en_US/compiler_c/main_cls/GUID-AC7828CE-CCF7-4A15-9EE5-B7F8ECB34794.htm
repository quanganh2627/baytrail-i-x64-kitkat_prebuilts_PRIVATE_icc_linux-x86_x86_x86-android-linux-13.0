Gz

Makes __stdcall the default calling convention.

IDE Equivalent

Windows: Advanced > Calling Convention

Linux: None

OS X: None

Architectures

IA-32 architecture

Syntax

Linux and OS X:

None

Windows:

/Gz

Arguments

None

Default

OFF

The default calling convention is __cdecl.

Description

This option makes __stdcall the default calling convention.
