Intrinsics to Determine Minimum and Maximum Operations

Parent topic: Vector Intrinsics
  • _mm512_[g]maxabs_ps/ _mm512_mask_[g]maxabs_ps
    Determines the absolute maximum of float32 vectors. Corresponding instruction is VGMAXABSPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_gmax_pd/ _mm512_mask_gmax_pd
    Determines the maximum of float64 vectors. Corresponding instruction is VGMAXPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_max_pd/ _mm512_mask_max_pd
    Determines the maximum of float64 vectors. On Intel® MIC Architecture, compare and jump instructions emulate this instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_gmax_ps/ _mm512_mask_gmax_ps
    Determines the maximum of float32 vectors. Corresponding instruction is VGMAXPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_max_ps/ _mm512_mask_max_ps
    Determines the maximum of float32 vectors. On Intel® MIC Architecture, compare and jump instructions emulate this instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_max_epi32/ _mm512_mask_max_epi32
    Determines the maximum of int32 vectors. Corresponding instruction is VPMAXSD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_max_epu32/ _mm512_mask_max_epu32
    Determines the maximum of uint32 vectors. Corresponding instruction is VPMAXUD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_gmin_pd/ _mm512_mask_gmin_pd
    Determines the minimum of float64 vectors. Corresponding instruction is VGMINPD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_min_pd/ _mm512_mask_min_pd
    Determines the minimum of float64 vectors. No corresponding instruction. On Intel® MIC Architecture, compare and jump instructions emulate this instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_gmin_ps/ _mm512_mask_gmin_ps
    Determines the minimum of float32 vectors. Corresponding instruction is VGMINPS. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_min_ps/ _mm512_mask_min_ps
    Determines the minimum of float32 vectors. No corresponding instruction. On Intel® MIC Architecture, compare and jump instructions emulate this instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_min_epi32/ _mm512_mask_min_epi32
    Determines the minimum of int32 vectors. Corresponding instruction is VPMINSD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_min_epu32/ _mm512_mask_min_epu32
    Determines the minimum of uint32 vectors. Corresponding instruction is VPMINUD. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
