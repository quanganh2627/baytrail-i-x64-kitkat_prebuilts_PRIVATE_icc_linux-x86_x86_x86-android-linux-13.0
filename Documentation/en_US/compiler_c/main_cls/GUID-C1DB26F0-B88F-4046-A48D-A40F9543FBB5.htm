_mm512_castpd_si512

Casts float64 vectors elements as integer vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __m512i __cdecl _mm512_castpd_si512(_m512d in);

Parameters

in

Vector used for casting operation

Description

Performs an element-by-element conversion/casting of float64 vector elements to integer vector elements without any change in value.

Returns

Returns the result of the casting operation.
