Appending Linker Options

This topic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

You may find it useful to customize the target code generation with the linker. You can do so by appending additional linker options to the built-in options.

To append additional linker options:

  • Use Eclipse* (C/C++ applications only):
    1. Open your project's property pages and select C/C++ Build > Settings > Intel® 64 C Linker.
    2. Select Miscellaneous.
    3. Enter the options in the Additional Options field.
  • Use the offload-option command line option.
