noBool

Disables the bool keyword.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

None

Windows:

/noBool

Arguments

None

Default

OFF

The bool keyword is enabled.

Description

This option disables the bool keyword.

Alternate Options

None
