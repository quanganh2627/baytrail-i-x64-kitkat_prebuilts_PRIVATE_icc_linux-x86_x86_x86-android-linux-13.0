_mm512_undefined_{ps|pd|epi32}

Makes all elements in a vector undefined. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __m512d __cdecl _mm512_undefined_pd(void);

extern __m512 __cdecl _mm512_undefined_ps(void);

extern __m512i __cdecl _mm512_undefined_epi32(void);

Parameters

None

Description

Converts all elements of a 512-bit vector to values that are not predefined.

In masked variants of KCi intrinsics, the values of   v1_old or v1 vectors may become irrelevant. In such cases, use one of the _mm512_undefined intrinsic in this topic to convert those irrelevant values of the v1_old or v1 vectors to newer undefined values.

Returns

Returns a 512-bit vector with all elements undefined.
