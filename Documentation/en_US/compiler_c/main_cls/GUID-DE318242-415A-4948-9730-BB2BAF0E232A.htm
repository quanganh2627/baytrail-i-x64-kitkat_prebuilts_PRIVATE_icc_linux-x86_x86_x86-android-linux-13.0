iwithprefixbefore

Similar to -iwithprefix except the include directory is placed in the same place as -I command line include directories.

IDE Equivalent

None

Architectures

IA-32, Intel® 64 architecture

Syntax

Linux and OS X:

-iwithprefixbeforedir

Windows:

None

Arguments

dir

Is the include directory.

Default

OFF

Description

Similar to -iwithprefix except the include directory is placed in the same place as -I command line include directories.

Alternate Options

None
