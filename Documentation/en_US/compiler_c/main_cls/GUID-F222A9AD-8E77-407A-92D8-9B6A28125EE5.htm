Intel® C++ Compiler XE 13.1 User and Reference Guides

Introduction

Intel® Cilk™ Plus, included in the Intel® C++ Compiler, allows you to improve performance by adding parallelism to new or existing C or C++ programs. This release provides support for building IA-32 and Intel® 64 architecture programs (32-bit and 64-bit) that run on the Microsoft Windows* OS, Linux* OS, and OS X*.

Most of the information pertains to all platforms; differences are noted in the text.

For a description of Intel® Cilk™ Plus features, see Getting Started.

Target Audience

Application developers who use Intel® Cilk™ Plus need at least a working knowledge of C or C++ programming: expert-level knowledge of C or C++ is optimal.
