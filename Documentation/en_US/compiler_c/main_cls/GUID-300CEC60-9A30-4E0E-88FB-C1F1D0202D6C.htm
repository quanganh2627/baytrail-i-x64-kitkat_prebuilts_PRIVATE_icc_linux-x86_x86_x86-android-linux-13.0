Guided Auto Parallelism Invocation dialog box

Use the Guided Auto Parallelism invocation dialog box to specify settings for Guided Auto Parallelism (GAP).

To access the Guided Auto Parallelism Invocation dialog box, select Intel Tools > Guided Auto Parallelism > Analyze File "hello_world.c".

Level of Analysis: Specify the desired level of analysis. Choose Simple, Moderate, Maximum or Extreme.

Suppress Remark #s: Specify one or more comma delimited remark identifiers to suppress.

Send remarks to a file: Check this box to redirect the analysis result into a file.

Remarks file: Enter the location of the file to redirect the analysis results.

Do not show again: Check this box to save the current settings as the default values. These settings are stored in the Guided Auto Parallelism preference page.
