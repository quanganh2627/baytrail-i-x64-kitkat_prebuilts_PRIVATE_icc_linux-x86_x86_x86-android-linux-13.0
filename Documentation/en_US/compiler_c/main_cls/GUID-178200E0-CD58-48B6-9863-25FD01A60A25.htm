Intel® C++ Compiler XE 13.1 User and Reference Guides

optimize

Enables or disables optimizations for code after this pragma till another optimize pragma or end of the translation unit.

Syntax

#pragma optimize("", on|off)

Arguments

The compiler ignores first argument values. Valid second arguments for optimize are given below.

off

disables optimization

on

enables optimization

Description

The optimize pragma is used to enable or disable optimizations. Specifying #pragma optimize("", off) disables optimization until either the compiler finds a matching #pragma optimize("", on) statement or until the compiler reaches the end of the translation unit.

Example

Example 1: Disabling optimization for a single function using the #pragma optimize

In the following example, optimizations are disabled for the alpha() function but not for omega().

#pragma optimize("", off)
alpha() {
...
}
#pragma optimize("", on)
omega() {
...
}

Example 2: Disabling optimization for all functions using the #pragma optimize

In the following example, optimizations are disabled for both the alpha() and omega() functions.

#pragma optimize("", off)
alpha() {
...
}
omega() {
...
}
