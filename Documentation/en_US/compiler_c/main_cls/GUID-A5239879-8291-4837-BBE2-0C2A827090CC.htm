Wport

Tells the compiler to issue portability diagnostics.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

None

Windows:

/Wport

Arguments

None

Default

OFF

The compiler issues default diagnostics.

Description

This option tells the compiler to issue portability diagnostics.

Alternate Options

None
