_mm512_set[r]4_pd

Sets elements of a float64 vector using specified values in the sequence, dcbadcba. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __m512d __cdecl _mm512_set4_pd(double d, double c, double b, double a);

extern __m512d __cdecl _mm512_setr4_pd(double a, double b, double c, double d);

Parameters

a

 float64 value

b

 float64 value

c

 float64 value

d

 float64 value

Description

Sets four values a, b, c, and d in a float64 result vector, in a specific sequence - dcbadcba, where the elements

v4, v0 = a

v5, v1 = b

v6, v2 = b

v7, v3 = d.

Returns

Returns a float64 vector with eight elements in a specific sequence - dcbadcba.
