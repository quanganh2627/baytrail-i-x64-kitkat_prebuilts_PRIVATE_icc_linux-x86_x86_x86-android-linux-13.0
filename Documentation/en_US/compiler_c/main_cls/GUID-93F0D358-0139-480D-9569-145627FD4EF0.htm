Intel® C++ Compiler XE 13.1 User and Reference Guides

Overview: Intrinsics across Intel Architectures

Most of the intrinsics documented in this section function across all supported Intel architectures. Some of them function across two Intel architectures.

The intrinsics are categorized as follows:

  • Integer Arithmetic Intrinsics

  • Floating-Point Intrinsics

  • String and Block Copy Intrinsics (only for IA-32 and Intel® 64 architectures)

  • Miscellaneous Intrinsics
