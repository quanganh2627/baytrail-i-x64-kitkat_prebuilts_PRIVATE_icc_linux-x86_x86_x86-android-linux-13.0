Overview: Intel® Streaming SIMD Extensions 4 (Intel® SSE4)

The intrinsics in this section correspond to Intel® Streaming SIMD Extensions 4 (Intel® SSE4) instructions. Intel® SSE4 includes the following categories:

  • Vectorizing Compiler and Media Accelerators. The prototypes for these intrinsics are in the smmintrin.h file.
  • Efficient Accelerated String and Text Processing. The prototypes for these intrinsics are in the nmmintrin.h file.
