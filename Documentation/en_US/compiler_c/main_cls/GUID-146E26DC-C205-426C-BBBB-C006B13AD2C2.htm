no-libgcc

Prevents the linking of certain gcc-specific libraries.

IDE Equivalent

None

Architectures

All

Syntax

Linux :

-no-libgcc

OS X:

None

Windows:

None

Arguments

None

Default

OFF

Description

This option prevents the linking of certain gcc-specific libraries.

This option is not recommended for general use.

Alternate Options

None
