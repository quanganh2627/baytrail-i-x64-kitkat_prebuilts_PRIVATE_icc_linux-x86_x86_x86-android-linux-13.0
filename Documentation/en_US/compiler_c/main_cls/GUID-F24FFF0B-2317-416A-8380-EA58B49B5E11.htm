Appending Archiver Options for Creating Libraries

This topic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

You may find it useful to customize the target code generation when creating libraries. You can do so by appending additional archiver options to the built-in options.

To append additional archiver options:

  • Use Eclipse* (C/C++ applications only):
    1. Open your project's property pages and select C/C++ Build > Settings > Intel® 64 Archiver.
    2. Select Archiver Command Line.
    3. Enter the options in the Archiver Options field.
  • Use the offload-option command line option.
