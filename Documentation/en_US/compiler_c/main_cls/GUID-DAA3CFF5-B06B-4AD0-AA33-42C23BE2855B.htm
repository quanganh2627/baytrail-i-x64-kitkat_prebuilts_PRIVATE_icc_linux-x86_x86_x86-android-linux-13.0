u (Linux* OS)

Tells the compiler the specified symbol is undefined.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-u symbol

Windows:

None

Arguments

None

Default

OFF

Standard rules are in effect for variables.

Description

This option tells the compiler the specified symbol is undefined.

Alternate Options

None
