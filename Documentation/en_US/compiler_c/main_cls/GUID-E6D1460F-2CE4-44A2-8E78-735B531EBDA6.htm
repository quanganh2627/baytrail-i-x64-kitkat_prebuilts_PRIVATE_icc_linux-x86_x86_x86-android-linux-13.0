dryrun

Specifies that driver tool commands should be shown but not executed.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

-dryrun

Windows:

None

Arguments

None

Default

OFF

No tool commands are shown, but they are executed.

Description

This option specifies that driver tool commands should be shown but not executed.

Alternate Options

None
