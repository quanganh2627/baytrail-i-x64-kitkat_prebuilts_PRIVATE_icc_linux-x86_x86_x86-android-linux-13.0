Reusing Guided Auto Parallelism Configurations

To save and reuse configuration settings for a project, folders, files, text range, or any combination thereof:

  1. Select the items you want to analyze.
  2. Right-click and select Intel Tools > Guided Auto Parallelism > Manage Analysis Configuration. The Manage Guided Auto Parallelism Analysis Selection table appears. You can change the default configuration settings by selecting an analysis configuration in the table and double-clicking Edit in the Parameters column.
  3. To save the current selections in a text file, select Save.
  4. To use a configuration you previously saved, select Intel Tools > Guided Auto Parallelism > Load Analysis Configuration and browse to the configuration file.
