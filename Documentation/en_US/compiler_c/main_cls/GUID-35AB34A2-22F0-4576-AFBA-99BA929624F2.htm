_mm512_setzero_{pd|ps|epi32}

Zeros all elements in a vector. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).

Syntax

extern __m512d __cdecl _mm512_setzero_pd(void);

extern __m512 __cdecl _mm512_setzero_ps(void);

extern __m512i __cdecl _mm512_setzero_epi32(void);

Parameters

None

Description

Sets all the elements of the 512-bit vector to zero.

Returns

Returns a 512-bit vector with all elements set to zero.
