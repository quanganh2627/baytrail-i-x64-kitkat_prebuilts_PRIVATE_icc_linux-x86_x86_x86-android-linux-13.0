Type Cast Intrinsics

Parent topic: Vector Intrinsics
  • Using Type Cast Intrinsics
  • _mm512_castpd_ps
    Casts float64 vectors elements as float32 vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_castps_pd
    Casts float32 vectors elements as float64 vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_castps_si512
    Casts float32 vectors elements as integer vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_castsi512_ps
    Casts integer vectors elements as float32 vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_castpd_si512
    Casts float64 vectors elements as integer vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
  • _mm512_castsi512_pd
    Casts integer vectors elements as float64 vector elements. There is no corresponding instruction. This intrinsic only applies to Intel® Many Integrated Core Architecture (Intel® MIC Architecture).
