_mm_cvtph_ps()

Converts four half-precision (16-bit) floating point values to single-precision floating point values.

Syntax

extern __m128 _mm_cvtph_ps(__m128i x, int imm);

Arguments

X

a vector containing four 16-bit FP values

Imm

a conversion control constant

Description

This intrinsic converts four half-precision (16-bit) floating point values to single-precision floating point values. The corresponding Intel® Post 32nm processor extension instruction is VCVTPH2PS.

Returns

A vector containing four single-precision floating point elements.
