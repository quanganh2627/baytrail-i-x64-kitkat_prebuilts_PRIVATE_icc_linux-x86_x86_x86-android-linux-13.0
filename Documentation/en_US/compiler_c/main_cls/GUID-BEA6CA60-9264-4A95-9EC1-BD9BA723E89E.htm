Overview: Eclipse* Integration

Eclipse* is an open source software development project dedicated to providing a robust, full-featured, commercial-quality, industry platform for the development of highly integrated tools. It is an extensible, open-source integrated development environment (IDE).

The Intel® C++ Compiler for Linux* OS provides an integration, also known as an extension, to the Eclipse* IDE. The Intel ® Debugger integration is included with the compiler installation, providing integration with Eclipse* and CDT.

CDT is a fully functional C/C++ for the Eclipse* IDE that allows developing, building, and running Intel® C/C++ projects in an visual, interactive environment. CDT is layered on Eclipse* and provides a C/C++ development environment perspective.

Note

Eclipse* and CDT are not bundled with the Intel® C++ compiler. They must be obtained separately.
