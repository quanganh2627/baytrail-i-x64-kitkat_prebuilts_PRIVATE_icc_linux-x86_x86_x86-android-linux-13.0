Ow

Tells the compiler to assume there is no cross-function aliasing.

IDE Equivalent

None

Architectures

All

Syntax

Linux and OS X:

None

Windows:

/Ow

/Ow-

Arguments

None

Default

OFF

The compiler assumes cross-function aliasing occurs.

Description

This option tells the compiler to assume there is no cross-function aliasing.

Alternate Options

None
