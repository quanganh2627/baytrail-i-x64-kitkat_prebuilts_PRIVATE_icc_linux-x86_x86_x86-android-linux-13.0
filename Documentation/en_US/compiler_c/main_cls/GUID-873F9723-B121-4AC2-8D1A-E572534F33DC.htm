Intel® C++ Compiler XE 13.1 User and Reference Guides

fblocks

Determines whether Apple* blocks are enabled or disabled.

IDE Equivalent

None

Architectures

IA-32, Intel® 64 architecture

Syntax

Linux :

None

OS X:

-fblocks

-fno-blocks

Windows:

None

Arguments

None

Default

-fblocks

Apple* blocks are enabled.

Description

This option determines whether Apple* blocks (block variable declarations) are enabled or disabled.

If you want to disable Apple* blocks, specify -fno-blocks.

To use this feature, OS X* or Mac OS* X 10.6 or greater is required.

Alternate Options

None
